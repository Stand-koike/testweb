# 林業GIS Pro

林業向けのWebベースGISアプリケーションです。OpenLayersを使用した地図表示・編集機能を提供します。

## 概要

- 地図上でのレイヤー管理
- ポリゴン・ライン・ポイントの描画・編集
- レイヤーの表示/非表示切り替え、並び替え
- ステータスバーによる操作状態の表示

## 技術スタック

- **HTML5** / **CSS3** / **JavaScript**
- [OpenLayers](https://openlayers.org/) 4.6.5（地図表示）
- [Font Awesome](https://fontawesome.com/) 6.4.0（アイコン）

## 使い方

1. リポジトリをクローンする
2. `index.html` をブラウザで開く（ローカルサーバーや `file://` で開けます）

```bash
# 例: 簡易サーバーで起動（Python 3）
python -m http.server 8000
# ブラウザで http://localhost:8000 にアクセス
```

## ライセンス

このプロジェクトのライセンスはリポジトリの設定に従います。
